# M&S Lets Fix It Services CC

Modern one-page website for M&S Lets Fix It Services CC, a motor vehicle repair and service centre in Boksburg, Gauteng.

## Deploy

This is a static website and can be imported directly into Vercel.

## Form

The enquiry form posts to FormSubmit and sends submissions to:

`letsfixitservices@gmail.com`

The first live submission may require email confirmation from FormSubmit before delivery is activated.
